01. Add a header component to `src/01_webapp/common/header/`.
02. The header stays at the very top (sticky header).

04. The header is wider than the inner context.

01. The header related files have been created:
	01-01. `src/01_webapp/common/header/header.component.ts`

	01-03. `src/01_webapp/common/header/header.scss`
02. The header has 2 sections:
	02-01. The left section (left aligned).
	02-02. The right section (right aligned).
03. The left section of the header shows a website link:
	03-01. Show a link like; `<a href="/"><span>[freshOffTheBot.com]</span></a>`.
04. The right section of the header shows a dropdown button (text-only).
	04-01. The button text is like; `<span>[Download App!]</span>`.
05. When the dropdown button is clicked, open the dropdown box.
	05-01. In the menu, show a description like; there are exclusive features in the native app, so we want to make the users download the native app.
	05-02. After that, show a download link like; `<a href="https://github.com/freshOffTheBot/based-translator/releases" target="_blank"><span>[Download App!]</span></a>`.

07. All the texts in the header use font size: `$font-size-500`.
	07-01. Use the font `$fontPixel`.
