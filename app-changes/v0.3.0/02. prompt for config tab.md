
# PROMPT FOR CONFIG TAB (v0.3.0)
- Add a config tab to show only necessary components.

## Add Tab Component

```md
Fren, you are a UI/UX master. I need your help to create a tab component.


## 1. Goal
- Add SCSS logic to the file `src/components/tab/tab.component.scss` for a tab component, and show the tab component in `src/dev/dev.html`.


## 2. Step-By-Step Steps
01. Understand the SCSS variables in `src/common/scss/app.variable.scss`.
02. Understand the SCSS components in `src/components/*.scss`:
  02-01. `src/components/typography/typography.component.scss`: text-related components SCSS file.
	02-02. `src/components/button/button.component.scss`: button component SCSS file.
	02-03. `src/components/card/card.component.scss`: card component SCSS file.
	02-04. `src/components/form/form.component.scss`: form component SCSS file.
03. Add a tab component in `src/components/tab/tab.component.scss`:
  03-01. Respect the existing SCSS code style.
  03-02. Compact + minimal + simple design.
04. Show the tab component in `src/dev/dev.html`:
  04-01. The secion: Components > Tab.
  04-02. Respect the existing style/design.
```
